      為減輕國人負擔及扶助企業，113年補助與貸款方案總整理！

      更新日期：民國113年5月20日
      主要資料來源：勞動部、財政部、經濟部、交通部觀光署

       

      因應疫情後的發展情勢，為減輕個人及家庭經濟負擔，並扶助企業、紓緩特定產業缺工情形，政府於民國(以下同)112年祭出各項疫情後之補助與獎勵措施，並新增113年紓困貸款及協助產業發展等方案，您可透過本文快速得知可申請的方案有哪些！請注意，以下內容為摘要說明，實際申請條件及金額請以各主管機關公告為準。

       

      目錄
      一. 補助及獎勵方案
      (一) 個人及家庭
      (二) 企業及團體
      二. 貸款方案
      三. 其他資訊

       

      一、補助及獎勵方案

       

      (一) 個人及家庭


      以下為政府各部會所推出之112年疫後補助方案（113年仍適用），並新增113年協助勞工安心就業方案（新方案以✨作為標示），同性質之補助從優擇⼀發給。

      適用對象 方案 補助說明 申辦方式 資料來源
      減班休息勞工 ✨僱用安定措施 於113年1月1日至113年6月30日期間，符合申請資格之勞工，其減班休息達30日以上，且為勞動部公告之7個行業，補貼其薪資差額之50%。 線上申請 勞動部勞動力發展署
      失業勞工 專案缺工就業獎勵 失業勞工經推介從事專案缺工工作「旅宿業房務(含清潔)人員、餐飲業內外場服務人員及廚務人員(不含廚師)，以及航空站地勤作業及航機清艙人員，詳情可參閱<職缺查詢>」，並達到獎勵資格後，可請領以下就業獎勵金，最長發給12個月，至113年6月30日截止：
      1. 一般勞工：每月6千元(全時)
      2. 特定對象：每月1萬元(全時)或5千元(部分工時)。
      ※ 工作地點位於偏遠(特定)地區，每月加給3千元(全時)或1千500元(部分工時)。      		 線上申請 勞動部
      專案缺工職業訓練試辦計畫-參訓獎勵 即日起至113年6月30日止，參加勞動部勞動力發展署各分署自辦、委辦或補助辦理之缺工工作相關職前訓練班，於結訓後可獲得參訓獎勵金，每人補助5千元，若訓練地點屬特定地區者，再加發3千元。 線上報名 勞動部勞動力發展署
      國保被保險人 國民年金保險費加碼補助 於114年10月31日(含)以前，繳納112年4月~12月國民年金保險費，可享有政府加碼補助50%自付保險費。
      ※ 低收入戶或重度以上身心障礙者，已由政府全額補助保險費。      		 免申請，由政府主動辦理 勞動部勞工保險局

       

      (二) 企業及團體

       

      以下彙整112年疫後所推動之企業補助措施（113年仍適用），並以✨標示113年最新方案內容，各方案若遇經費用罄將提前截止，請符合資格之企業儘早提出申請：

      適用對象 方案 補助說明 申辦方式 資料來源
      製造業 ✨113年中小型製造業(≤9人)低碳及智慧化升級轉型補助 1. 受理期間：至經費用罄為止。
      2. 適用企業：經常性雇用員工數9人以下之中小企業。
      3. 補助標的：企業進行產業低碳化及智慧化升級。
      4. 補助上限：最高300萬元。      		 線上申請 經濟部中小及新創企業署
      ✨113年中小型製造業(≥10人)低碳及智慧化升級轉型個案補助 1. 受理期間：至經費用罄為止。
      2. 適用企業：經常性雇用員工數10人以上之中小企業。
      3. 補助標的：企業進行產業低碳化及智慧化升級。
      4. 補助上限：最高500萬元。      		 線上申請 經濟部產業發展署
      ✨以大帶小製造業低碳及智慧化升級轉型補助 1. 受理期間：至經費用罄為止。
      2. 適用企業：中心廠須符合經常性雇用員工數10人以上。
      3. 補助標的：由中心廠帶動其供應鏈廠商，共同推動低碳化或智慧化轉型。
      4. 補助上限：低碳化最高補助3千萬元、智慧化2千萬元。      		 線上申請 經濟部產業發展署
      ✨113年納管工廠低碳及智慧化基礎轉型個案補助 1. 受理期間：至113年11月29日止，或至經費用罄為止。
      2. 適用企業：為納管工廠，並經地方政府核定改善計畫。
      3. 補助標的：工廠生產及防災、污染防治管理之數位化或低碳化轉型。
      4. 補助上限：最高150萬元。      		 線上申請 經濟部產業發展署
      中小企業 ✨協助中小企業出口及會展減碳 為協助中小企業出口拓銷及減碳，113年推出5項輔導專案，包含輔導企業智慧科技行銷、建構數位能力開發海外市場、出口產減碳包裝設計，協助加工食品業海外通路行銷，以及輔導會展業者落實減碳。詳細申請資格及受理期間，請依各措施規定為準。 線上申請 經濟部國際貿易署
      旅行業 加速擴大吸引國外觀光客方案 我國立案綜合或甲種旅行社與境外組團旅行社合作，促進外籍旅客來臺旅遊，其組團人數達4人以上，且停留天數為3天2夜以上，且來臺旅遊期間在114年6月30日前，可申請獎助金5千元~5萬元不等，其獎助金分配比例為國內旅行社40%、境外組團旅行社60%。 線上申請 交通部觀光署
      旅宿業 補助旅宿業穩定接待國際旅客服務量能方案 即日起至114年5月31日止，符合申請資格之合法旅館及民宿，新聘僱之房務及清潔服務人員於112年4月1日至113年3月31日到職，並連續在職6個月以上，且薪資符合規定者，補助每人每月5千元，最長12個月。每家旅館或民宿其申請補助總人數，以房間數1/8為限。 線上申請 交通部觀光署
      民營事業單位、民間團體、高中(職)或大專校院 專案缺工職業訓練試辦計畫-先僱後訓 即日起至113年6月30日止，符合申請資格之用人單位，於臺灣就業通公告「專案缺工工作」，僱用後進行工作崗位訓練，補助每人每月最高1萬2千元，訓練地點屬特定地區，每人每月最高補助1萬5千元，最長3個月。 線上申請 勞動部勞動力發展署

       

       

       

      二、貸款方案

       

      以下彙整112年疫後所推動之企業及個人貸款方案（113年仍適用），113年新方案則以✨作為標示：

      適用對象 方案 貸款說明 申辦方式 資料來源
      勞保被保險人 ✨113年勞工保險被保險人紓困貸款
      (已截止)      		 參加勞工保險年資滿15年且符合貸款條件者，可於113年1月19日(含)前提出貸款申請。每人最高可貸新臺幣(以下同)10萬元，貸款期間為3年，前6個月按月付息不還本，年息為2.04%（並於每年1月及7月公告調整）。 線上申請
      (已截止)      		 勞動部勞工保險局
      中小企業 疫後振興專案貸款 1. 受理期間：至114年10月31日止。
      2. 適用企業：須符合原申貸紓困振興貸款、疫後時期營運困難或停業再出發(任一)。不含金融及保險業、特殊娛樂業。
      3. 貸款用途：購建(修)廠房或營運場所、購置機器設備或營運週轉金。
      4. 貸款額度：最高3,500萬元，利率為2.095%（員工數未達5人之中小企業，或僅有稅籍登記之營利事業至多可貸400萬元。若貸款金額在100萬元以下，其利率為1.595%。）
      5. 自112年10月起無利息補貼，但仍受理申貸。      		 請洽<承貸金融機構>，或進行<資格認定暨輔導申請> 經濟部中小及新創企業署
      低碳智慧納管貸款 1. 受理期間：至114年10月31日止。
      2. 適用企業：朝低碳化及智慧化轉型企業、取得特定工廠登記之業者或納管工廠(任一)。不含金融及保險業、特殊娛樂業。
      3. 貸款用途：購建(修)廠房或營運場所、購置機器設備或營運週轉金。
      4. 貸款額度：資本性支出不超過計畫經費80%，週轉性支出最高3,500萬元，利率為2.095%。
      5. 自112年10月起無利息補貼，但仍受理申貸。
      中小企業千億振興融資方案 1. 受理期間：至113年12月31日止。
      2. 適用企業：微型及新創事業、具研發創新能力之中小企業。
      3. 貸款用途：用於營運週轉、設備升級、研發創新產品或技術、研發人才培育。
      4. 貸款額度：由各銀行依個案實際需求核定額度，貸款利率最高不超過中華郵儲2年期定儲機動利率+1%。      		 請洽<承貸銀行> 財政部國庫署
