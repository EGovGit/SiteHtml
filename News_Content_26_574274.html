      家中遭逢變故，政府提供哪些急難紓困方案？

      更新日期：民國113年4月16日
      主要資料來源：教育部、文化部、內政部、原住民族委員會、僑務委員會、國軍退除役官兵輔導委員會、各縣市政府

       

      若您因為家族成員傷病、死亡、失業、失蹤、服兵役、入獄服刑或財產遭凍結等因素，或本人遭遇重大事件，導致生活陷於困境，政府針對遭逢一時急難的民眾，提供相關救助方案。另外，您的身分可能同時符合多個方案，除非法令沒有限制，否則不得重複領取政府其他相同性質之補助。

       

      目錄
      一. 急難救助方案
      二. 其他相關補助方案或資源
      三. 各縣市資訊

       

      一、急難救助方案

       

      身分別 方案 概述 給付項目 (單位：新臺幣/元) 去哪辦？
      一般民眾 急難救助 若本人或家中成員遭遇<社會救助法第21條>之情況，可提出申請，實際補助內容依<👇各縣市規定>為準。 1. 喪葬救助：2千~3萬元
      2. 傷病救助：2千~2萬元
      3. 生活救助：2千~2萬元
      4. (川)車資救助：部分縣市提供客運或火車車資返鄉      		 戶籍地公所
      急難紓困 針對本人或家中成員死亡、重傷病，或因故無法工作而導致生活陷困；或因經濟因素為脆弱家庭成員或為自殺通報個案等。 救助金1~3萬元，若為急迫性個案可先發給5千元
      災害救助 補助因水災、風災、地震、土石流等天然災害，導致損害重大，生活受到影響者，實際補助內容<👇各縣市規定>為準。 1. 死亡及失蹤救助：20萬元/人
      2. 重傷救助：10萬元/人
      3. 安遷救助：每戶2萬元/人，以5人為限
      4. 住宅淹水救助：5千~2萬元/戶
      5. 住宅土石流救助：1萬~2萬元/戶
      6. 住屋(財物)損毀救助：1萬5千~2萬元/戶      		 災害地公所
      學生及幼兒 學產基金設置急難慰問金 針對年齡未滿25歲之在學學生及幼兒，但不包含就讀碩/博士班、空中進修學院、空中大學及其附設專科部。 1. 本人傷病住院或意外死亡：1萬元 (重大傷病2萬元)
      2. 遭父母或監護人虐待、遺棄等：2萬元
      3. 父母或監護人有以下情形：
      (1) 失蹤、入獄服刑、失業：1萬元
      (2) 重大傷病或死亡：2萬元
      (3) 因災受傷住院：5千或1萬元
      4. 因家境特殊或其他重大意外事故等，以專案申請。      		 學校、幼兒園
      役男 一次安家費及三節扶助金 役男入營服兵役期間，其家屬不能維持生活(符合列級家屬)，政府提供各項<扶助措施>。其餘未列級之替代役役男、常備兵役及義務役軍人之慰問金，可參閱<役男權益>。 依核列等級發給：
      1. 甲級：21,390元~71,300元/口
      2. 乙級：12,834元~42,780元/口
      3. 丙級：6,417元~21,390元/口      		 戶籍地公所
      急難慰助金、
      喪葬補助費      		 因天災人禍發生以下狀況：
      1. 房屋倒塌：半倒1萬5千元、全倒3萬元
      2. 房屋淹水：1萬~2萬元
      3. 農田或魚塭埋沒流失：2千~1萬2千元
      4. 重傷：2萬
      5. 死亡或失蹤：6萬元 (另有喪葬補助費2萬5千元)
      榮民(眷) 榮民(眷)、遺眷急難救助及慰問 榮民(眷)或遺眷死亡、意外、重病、遭逢重大事故，或因故無法工作等，導致生活陷困者，可申請救助。 1. 意外傷害：1萬或2萬元
      2. 重病或其他原因無法工作：1萬或2萬元
      3. 死亡殮葬：最高2萬元
      4. 返鄉車資救助 (限榮民)：最高1千元
      5. 就養榮民亡故，發給無職業遺眷救助金：3萬元
      6. 遭逢重大事故：最高5萬元      		 居住地<榮民服務處>
      原住民 原住民急難救助 針對本人或家中成員死亡、重傷病、因故無法工作或遭受重大災害等，而導致生活陷困者。 1. 死亡救助：最高1萬~2萬元
      2. 醫療補助：最高1萬~2萬元
      3. 生活扶助：最高1萬元
      4. 重大災害救助：最高1萬~5萬元      		 戶籍地公所、原住民事務委員會
      新住民
      (設籍前)      		 設籍前新住民遭逢特殊境遇家庭扶助 補助新住民因故導致生活陷困、遭遇配偶遺棄、為暴力受害者等，或者配偶死亡或失蹤，實際補助內容依<👇各縣市規定>為準，另可參閱內政部移民署<新住民發展基金>。 1. 緊急生活扶助：該縣市最低生活費3個月
      2. 15歲以下子女生活津貼：補助最低工資*10%/月
      3. 未滿6歲子女托育津貼：1千5百元/月
      4. 傷病醫療：補助自付醫療費超過3萬元的部分，最高12萬元/年
      5. 法律訴訟：最高5萬元/年
      6. 返鄉往返機票：最高2萬~3萬元/年      		 配偶或子女之戶籍地公所
      僑胞 僑胞醫療急難及喪葬慰問 僑胞回國觀光或探親時，因傷病、死亡或災害等突發事件遭受重大損失，可申請慰問金。 1. 住院醫療：1千~1萬元
      2. 天災人禍：1千~1萬元
      3. 死亡：1萬元      		 僑務委員會
      從事文化工作者 文化部績優文化人士急難補助(113年度待公告) 績優文化人士若發生重大變故時，給予本人或家屬關懷慰問。 1. 急難補助：因重大變故導致生活陷困，最高10萬元
      2. 急難慰問：因意外或重大疾病導致傷殘或亡故，1萬元 (家境清寒最高10萬元)      		 郵寄至文化部

      二、其他相關補助方案或資源

       

      身分別 方案 概述 給付項目 (單位：新臺幣/元) 去哪辦？
      一般民眾 實(食)物銀行 針對經濟弱勢家庭、領有社會福利仍陷困者、本人或家中成員因故致生活陷於困境申請，詳情依<👇各縣市規定>為準。 1. 政府媒合各界捐贈之物資，經由專業社工評估個案需求後，發放生活物資。
      2. 或提供民眾食物券及生活券，至合作店家領取熱食、生鮮食材及日常生活必需品等。      		 居住地公所
      失業勞工 失業勞工子女就學補助 育有就讀高中職或大專校院子女之失業勞工(包含已就業，但前一年內失業，且就業未超過3個月)，且與配偶在前一年度綜合所得總額在新臺幣148萬元以下，可提出申請。 1. 高中職 (含五專前3年)學生：公立4千元/私立6千元。
      2. 大專校院 (含五專後2年)學生：公立1萬3千600元/私立2萬4千元。
      👉 申請人符合獨力負擔家計或其子女就讀大專校院有2人以上，依上述補助金額加給20%。      		 1. 勞保局各辦事處
      2. 或線上申辦

      👉 關於「弱勢兒童及少年」的補助資訊，可參閱【弱勢兒少福利報您知，給孩子一個明亮的未來！】。
      👉 關於「失業者」的補助資訊，可參閱【失業該怎麼辦？4大失業相關補助一次整理給你看！】。
      👉 若您想了解「處理身後事」之喪葬津貼及補助，可參閱【處理身後事，帶您看喪葬津貼及補助有哪些？】。

      站內相關服務連結：

      • 急難救助申請 符合社會救助法第21條規定者，可向戶籍地之鄉(鎮市區)公所及縣(市)政府，提出急難救助申請。
      • 僑胞醫療急難及喪葬慰問金 海外僑胞回國定居、觀光或探親時，遇有特殊急難事件，如傷病住院醫療或死亡等，本人或家屬可向僑務委員會申請慰問金。
      • 臺中市-急難救助申請 設籍臺中市之民眾，因家庭發生重大變故，於事實發生3個月內向戶籍所在地區公所提出申請。
      • 臺中市-市民醫療補助申請設籍臺中市，並列冊(中)低收入戶，或符合經濟弱勢兒少、特殊境遇家庭、身心障礙者、生活陷困或其他特殊情形等，可申請市民醫療補助，每人每年以新臺幣15萬元為限。
