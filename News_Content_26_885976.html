      遭遇性騷擾，我該怎麼辦？ 要如何申訴？

      更新日期：民國115年7月24日
      主要資料來源：勞動部、內政部警政署、教育部、衛生福利部、法務部

       

      本文將說明各類型性騷擾的定義及適用範圍，例如遭受跟蹤騷擾、勞工遭遇職場性騷擾、學生遭遇校園性騷擾，或在公共場合被陌生人性騷擾等，並摘述民眾在遭受性騷擾時該如何應對、申訴途徑及協助資源有哪些，協助您安心工作、上學及生活！請注意，以下內容供民眾參考，詳情請以各主管機關最新的公告為準。

       

      目錄
      一. 遭遇性騷擾，我該怎麼辦？該向誰申訴？
      二. 被跟蹤騷擾
      三. 遭遇職場性騷擾
      四. 遭遇校園性騷擾
      五. 遭遇校園及職場以外的性騷擾

       

      一、遭遇性騷擾，我該怎麼辦？該向誰申訴？

       

      (一) 遭遇性騷擾，我該怎麼辦？

      以下建議供您參考：

      • • 不要忽視自己的感受：雖然每件事不一定構成性騷擾，但請相信自己的直覺，當您感到不安或受到冒犯時，請不要忽視自己的感受。
      • • 尋求情緒支持：向信任的親友說出自身經歷與感受，不要因此感到自責或羞愧。
      • • 向性騷擾者(以下簡稱行為人)說不：除了口頭說不，建議您也可以使用電子郵件或通訊軟體向行為人要求停止性騷擾，並訴說自己的感受。
      • • 當下尋求協助：如果正被人跟蹤騷擾或性騷擾，請試著保持冷靜，並尋求周邊的人幫忙。
      • • 記錄人事時地物：詳細記錄行為人特徵、事件發生的細節、時間及地點、周遭在場的其他人，以及物證留存。
      • • 蒐集各項證據，以作為日後申訴用：例如在場證人、監視器畫面、通訊軟體上所留的文字、圖片、影像紀錄或電子郵件內容等等。

       

      (二) 該向誰申訴？

       

      以下為性騷擾案件處理原則，若您仍不確定應該向哪個機關申訴，建議您可就近向所在地社會局(處)、行為人所屬學校或教育局(處)、勞工局(處)及警察局任一單位提出申訴。

      本文也為您彙整各類型性騷擾案件常見問答，幫助您快速了解適用法規、申訴管道及相關權益，點此快速看👇被跟蹤騷擾 / 遭遇職場性騷擾 / 遭遇校園性騷擾 / 遭遇校園及職場以外的性騷擾。

       

      圖片出處 : 勞動部就業平等網 民眾遭受性騷擾之申訴案件移送管轄機關標準作業流程

       

       

       

      二、被跟蹤騷擾

       

       

      💡什麼是跟蹤騷擾？

      • 依據跟蹤騷擾防制法第3條第1項規定，以人員、車輛、工具、設備、電子通訊、網際網路等各種方式，針對「特定的人」，做出「反覆或持續」與性或性別有關之以下任一行為，且違反其意願，使人感到不舒服或恐懼，足以影響其日常生活：

      • • 監視、觀察或跟蹤：例如透過望遠鏡監視、GPS定位追蹤、在遠處跟隨，或打聽其行程等。
      • • 盯梢、守候或接近：在被害人住家、工作地、學校或經常出入的場所附近徘徊或蹲點。
      • • 威脅、辱罵、歧視或貶抑：以言行或文字進行敵視或負面言論，例如當面或在社群媒體上對其外貌或性別進行羞辱。
      • • 通訊干擾：利用電話、傳真、社群媒體等，持續地撥打電話或傳送大量訊息。
      • • 不當追求：被害人已明確拒絕，行為人仍持續做出讓人感到困擾的追求行為。
      • • 寄送物品：寄送或留下物品(例如花束、禮物、信件或垃圾)，或公開播放影片。
      • • 妨害名譽：散布不實消息、私密照或其他有害名譽的事物。
      • • 冒用個資：未經同意利用個資代訂貨物或服務，使被害人感到困擾。例如代訂外送或網購送到家門口。

      💡該向誰申訴？

      • 請向所在地警察局報案，請警察機關(以下簡稱警方)對行為人核發告誡書，必要時也可請求警方採取保護措施，例如增設巡邏箱、護送返家等。被害人也可提出跟騷罪告訴，追究行為人的刑事責任。

      💡若行為人再犯，該怎麼辦？

      • 行為人經警方書面告誡後2年內再犯，被害人可向法院聲請保護令。有關保護令辦理流程、書狀範例等資訊，可參閱司法院跟蹤騷擾保護令。

      💡如果遭遇單次偶發的跟蹤，是否成立跟騷罪？

      • 跟騷罪的基本核心要件為「反覆或持續」，若僅發生1次，通常不構成跟騷罪。但若涉及不當接觸或性語言，則屬於性騷擾防治法第2條第1項第1款及第2款規定，可參閱👇【五. 遭遇校園及職場以外的性騷擾，我該怎麼辦？】

      💡我有什麼資源可以運用？

      • • 113保護專線：24小時免付費專線，受理全國家庭暴力、老人保護、身心障礙者保護、兒童少年保護及性侵害、性騷擾事件通報或求助諮詢。
      • • 113線上諮詢：或可傳簡訊至113，或使用線上諮詢的服務。
      • • 110報警：若有立即危及人身安全事件，請直接撥打報警電話。
      • • 警政服務APP：可使用此App的GPS定位功能，回傳自身位置，在遭遇危急狀況時可立刻報案，以利警方掌握被害人的行動軌跡。
      • • 財團法人法律扶助基金會：被害人若有法律扶助的需求，經審核符合資格者(如中低/低收入戶、特殊境遇家庭等)，即可獲得扶助。

      💡我想知道更多有關跟蹤騷擾相關資訊，可以去哪裡查？

      • 如果您想了解更多詳情，像是跟蹤騷擾行為種類、申訴流程，以及行為人約制法規等，可參閱內政部警政署跟蹤騷擾防制法懶人包及臺灣高等檢察署跟蹤騷擾防制法之介紹。

      三、遭遇職場性騷擾

       

       

      💡什麼是職場性騷擾？

      • 依據性別平等⼯作法第12條第1項至第3項規定，以下為簡單說明：

      • • 受僱者在執行職務時，任何人以性要求、具有性意味或性別歧視之言行(例如講黃色笑話、不當肢體碰觸、傳送不雅照片)，對其造成敵意性、脅迫性或冒犯性之工作環境，致侵犯或干擾其人格尊嚴、人身自由或影響其工作表現。
      • • 雇主對受僱者或求職者為明(暗)⽰之性要求、具有性意味或性別歧視之⾔詞或⾏為，作為勞務契約成立、存續、變更或分發、配置、報酬、考績、陞遷、降調、獎懲等之交換條件。
      • • 對於因僱用、求職或執行職務關係受自己指揮、監督之人，利用權勢或機會為性騷擾。
      • • 其他適用性別平等工作法規定之性騷擾：
        (1) 受僱者於非工作時間，遭受所屬事業單位之同一人，為持續性性騷擾。
        (2) 受僱者於非工作時間，遭受不同事業單位，具共同作業或業務往來關係之同一人，為持續性性騷擾。
        (3) 受僱者於非工作時間，遭受最高負責人或僱用人為性騷擾。

      💡職場性騷擾與一般的性騷擾有什麼不同?

      • • 職場性騷擾是指在職場中，因執行職務時所遭到與性或與性別有關的騷擾行為。
      • • 職場性騷擾與一般通稱的性騷擾行為並沒有不同，只是該性騷擾行為發生的地點在職場，基於督促雇主保障勞工權益及營造平等友善的工作環境，適用性別工作平等法來規範雇主善盡防治與處理的義務與責任。

      💡執行職務時，遭遇合作廠商、客戶或其他有業務往來公司的員工性騷擾，該怎麼辦？

      • 因雙方雇主同負性騷擾防治義務，可向雇主提出申訴，由雇主依法通知他方雇主共同協商解決，詳情請參閱執行職務時遭遇具共同作業或業務往來關係的不同公司員工性騷擾怎麼辦？。

      💡執行職務時，遭遇不特定人性騷擾，該怎麼辦？

      • • 何謂遭遇「不特定人」性騷擾？像是公務人員遭受洽公民眾性騷擾、醫事人員在醫院遭受病患或家屬性騷擾、商場人員在營業場所遭到顧客性騷擾、社群網站小編遭受留言者性騷擾等等。
      • • 該怎麼辦？被害人向雇主反映，雇主須採取立即有效之糾正及補救措施，依被害⼈需求提供諮詢及其他必要服務，並協助被害人提出申訴。
      • • 如何申訴？當行為人不明，向性騷擾事件發生地之警察局提出申訴；行為人有所屬政府機關(構)、部隊、學校，請向該政府機關(構)、部隊、學校提出申訴。
      • • 詳情可參閱執行職務時遭遇不特定人性騷擾怎麼辦？及👇【五. 遭遇校園及職場以外的性騷擾，我該怎麼辦？】。

      💡若是性騷擾行為不在執行職務的時候，像是發生在下班之後，被同事或客戶性騷擾，屬於職場性騷擾嗎？

      • 若該性騷擾⾏為僅發生在非執⾏職務的時候，仍可依性騷擾防治法規定提出申訴。若該性騷擾⾏為不僅在非執⾏職務的時候發⽣，也在執行職務期間遭受同一個人性騷擾，仍屬於職場性騷擾。

      💡職場性騷擾的申訴管道有哪些？

      • • 向雇主申訴：以言詞、書面或電子郵件方式，向雇主或公司受理性騷擾申訴之人員或單位提出申訴。雇主須於接獲申訴之翌日起2個月內調查並結案，但必要時可延長1個月。
      • • 向地方勞工行政主管機關申訴：若有以下情形，可向地方勞工行政主管機關申訴。
        (1) 受僱者或求職者遭受老闆（最高負責人或僱用人）性騷擾。
        (2) 經向雇主申訴，雇主未處理。
        (3) 不服雇主所為調查或懲戒結果。
      • • 向警察局申訴：警察局製作調查筆錄，若屬於職場性騷擾事件，將移送地方勞工行政主管機關處理；若屬於跟蹤騷擾行為，可請警方對行為人核發告誡書，或被害人也可向行為人提出跟騷罪告訴，可參閱👆【二. 被跟蹤騷擾】。

      💡我必須在什麼時間之內向地⽅勞⼯行政主管機關提申訴？

      • • 行為人非具權勢地位：自知悉性騷擾時起，逾2年提起者，不予受理；自該行為終了時起，逾5年者，亦同。
      • • 行為人具權勢地位：自知悉性騷擾時起，逾3年提起者，不予受理；自該行為終了時起，逾7年者，亦同。
      • • 行為人為最高負責人或僱用人：申訴人得於離職之日起1年內申訴。但自該行為終了時起，逾10年者，不予受理。但如依前開規定而有較長申訴期限者，依前開規定辦理。
      • • 性騷擾發生時，被害人為未成年，得於成年之日起3年內申訴。但如依前開規定而有較長申訴期限者，依前開規定辦理。

      💡我有什麼資源可以運用？

      • 勞動部彙整各地方主管機關有關工作場所性騷擾被害人可運用之相關資源，依縣市政府分類查詢，服務類型包含法律諮詢、精神醫療及驗傷採證、心理諮商、社會福利資源等，有關申請方式及服務內容，民眾可洽各縣市政府，詳請可參閱縣市政府提供工作場所性騷擾被害人之相關資源。

      💡我想知道更多有關職場性騷擾相關資訊，可以去哪裡查？

      • 如果您想了解更多詳情，像是性騷擾申訴流程、發生情境、相關法令、雇主應盡防治義務，以及被害人可以主張的民、刑事責任等，可參閱勞動部職場性騷擾預防及處理。

      四、遭遇校園性騷擾

       

       

      💡什麼是校園性騷擾？

      • 依據性別平等教育法第3條第3項規定，發生在學校校長、教師、職員、工友或學生其中之㇐，同時另㇐方為學生，且有以下任一情形：
      •  
      • • 以明(暗)示之方式，從事不受歡迎且與性或性別有關之言詞或行為，以致影響他人生活。
      • • 以性或性別有關之行為，並以此作為影響對方的學習或工作權益(例如成績、工作表現等)。

      💡除了性騷擾，校園性別事件還包含哪些？

      • 依據性別平等教育法第3條第3項規定，校園性別事件還包含以下情形：
      •  
      • • 性侵害：以強暴、脅迫、恐嚇、催眠術或其他違反其意願之方式而為性交或猥褻，或以詐術使人誤信為自己配偶，而聽從其為性交者。詳情可參閱妨害性自主罪。
      • • 性霸凌：透過語言、肢體或其他暴力，對於他人之性別特徵、性別特質、性傾向或性別認同進行貶抑、攻擊或威脅之行為，且非屬性騷擾者。
      • • 校長或教職員工違反與性或性別有關之專業倫理行為：指校長或教職員工與未成年學生發展親密關係，或利用不對等之權勢與學生發展以性行為或情感為基礎等有違專業倫理之關係。

      💡學生在校外實習，於實習期間遭受性騷擾，屬於校園性騷擾嗎？

      • 若性騷擾行為發生在校外實習期間，且對象非實習指導人員時，則屬於性別平等工作法第2條第5項規範，請向雇主或地方勞工行政主管機關申訴，可參閱👆【三. 遭遇職場性騷擾，我該怎麼辦？】。但對象為實習指導人員時，則屬於校園性別事件。

      💡若行為人為校外人士(例如房東、店家等)，學校是否受理？

      • 學校應協助被害學生，依性騷擾防治法第14條第3項規定，向行為人所屬機關(構)、部隊、學校、雇主或當地主管機關提出申訴；若行為人不明時，請向事件發生地之警察局提出申訴。可參閱👇【五. 遭遇校園及職場以外的性騷擾，我該怎麼辦？】

      💡校園性別事件的申訴管道有哪些？

      • • 向學校申請調查或檢舉：被害人、其法定代理人或實際照顧者，以書面方式向行為發生時行為人所屬之學校(學務處、教導處或學校指定專責單位等)申請調查或檢舉。或學校完成通報後，由學校性別平等教育委員會評估案情自行檢舉。
      • • 向學校所屬主管機關申請調查或檢舉：行為人現為或曾為學校校長者，應向行為發生時之學校所屬主管機關（教育部、當地縣市政府教育局/處）申請調查或檢舉。

      💡學校該有何動作？

      • • 學校校長及教職員工獲知疑似校園性別事件後，應在24小時內，向學校及當地主管機關通報。
      • • 針對未滿18歲之兒少校園性別事件，校方須至關懷e起來、當地家庭暴力及性侵害防治中心或社會局/處通報。

      💡我可以獲得什麼協助？

      • 事件管轄學校或機關，於必要時，應提供被害人心理諮商與輔導、法律協助、課業協助、經濟協助、社會福利資源轉介服務，或其他必要之保護措施或協助。

      💡我想知道更多有關校園性騷擾相關資訊，可以去哪裡查？

      • 如果您想了解更多詳情，像是校園性別事件通報及申訴流程、相關法令、線上課程等，可參閱教育部性別平等教育全球資訊網校園性別事件處理流程。

      五、遭遇校園及職場以外的性騷擾

       

       

      💡什麼是性騷擾？

      • 除性侵害以外，對他人實施違反其意願而與性或性別有關之行為，且有以下任一情形，詳情請參閱性騷擾防治法第2條第1項第1款及第2款規定：

      • • 以言語、行為或其他方式，對他人做出不當言行，使對方感到尷尬、害怕、不舒服、被冒犯或受到羞辱，或影響到他人工作、求學或日常生活。
      • • 以性或性別相關的要求作為交換條件，例如要求對方接受約會、肢體碰觸或其他不當要求，否則可能影響其升遷、成績、錄取資格、訓練機會或其他權益等。

      💡什麼是權勢性騷擾？

      • 利用職務、身分或權力上的優勢，對於受自己管理、指導或監督的人進行性騷擾。

      💡該向誰申訴？

      • • 向政府機關(構)、部隊、學校申訴：申訴時行為人有所屬政府機關(構)、部隊、學校。
      • • 向縣市主管機關申訴：申訴時行為人為政府機關(構)首長、各級軍事機關(構)及部隊上校編階以上之主官、學校校長、機構最高負責人或僱用人。
      • • 向警察局申訴：行為人不明或非屬上述2種情況者。

      💡我必須在什麼時間之內提出申訴？

      • • 行為人不具權勢地位：自知悉事件發生後2年內提出申訴。但自性騷擾事件發生之日起逾5年者，不得提出。
      • • 行為人具權勢地位：自知悉事件發生後3年內提出申訴。但自性騷擾事件發生之日起逾7年者，不得提出。
      • • 事件發生時被害人為未成年：於成年後3年內提出申訴。但如依前開規定而有較長申訴期限者，依前開規定辦理。

      💡我有什麼資源可以運用？

      • • 113保護專線：24小時免付費專線，受理全國家庭暴力、老人保護、身心障礙者保護、兒童少年保護及性侵害、性騷擾事件通報或求助諮詢。
      • • 113線上諮詢：或可傳簡訊至113，或使用線上諮詢的服務。
      • • 110報警：若有立即危及人身安全事件，請直接撥打報警電話。
      • • 各縣市政府性騷擾聯絡窗口：可致電詢問有關性騷擾申訴相關行政程序。
      • • 財團法人法律扶助基金會：被害人若有法律扶助的需求，經審核符合資格者(如中低/低收入戶、特殊境遇家庭等)，即可獲得扶助。

      💡我想知道更多有關性騷擾防治相關資訊，可以去哪裡查？

      • 如果您想了解更多詳情，像是性騷擾申訴處理原則、求助資源，可參閱衛生福利部性騷擾防治及臺灣高等檢察署性騷擾防治專區。
