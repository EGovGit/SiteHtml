      保障產權好輕鬆！稅籍異動、地籍異動即時通申請說明一次看

      更新日期：民國114年9月22日
      主要資料來源：財政部、內政部地政司、各縣市地方政府

       

      當房價不斷持續攀升，不動產成為許多家庭最重要的資產，卻也吸引詐騙集團虎視眈眈，稍有不慎，就可能遭到偽冒過戶或受騙抵押。為保障不動產所有權人的安全，內政部自105年10月31日實施「地籍異動即時通」 服務，協助民眾即時掌握不動產登記的變動情況，藉此防範詐騙與偽冒申辦；基於不動產在過戶登記前，必須先向地方稅捐稽徵機關申報土地增值稅及契稅，財政部於114年7月1日推出「稅籍異動即時通」服務，在產權異動進入地政程序前就先發出警示，形成防詐的第一道防線。本文將帶您了解兩項服務申請資格及流程，幫助您快速上手、輕鬆守護財產！

      一、稅籍異動即時通

       

      為讓民眾掌握不動產異動資訊、預防詐騙冒用或不當移轉情形，財政部推出免費的稅籍異動即時通服務。當不動產所有人名下的房屋或土地發生契稅或土地增值稅申報時，稅籍異動即時通會在案件「收件」時即時發送手機簡訊、電子郵件或LINE推播通知（需綁定進階會員），協助民眾第一時間察覺異常異動，守護財產安全。

       

      (一) 申請資格

       

      稅籍異動即時通應由本人申請，以下情況可例外由法定代理人代為申請。

      • 未成年人：由法定代理人（父母）申請，須提出法定代理證明文件。
      • 受監護宣告者：由法定代理人（監護人）申請，須提出法定代理證明文件。

       

       

      原則上申請人即使名下無土地或建物，也可預先申請稅籍異動即時通服務；但因各縣市作業流程有細微差異，例如嘉義市推出「稅籍」及「地籍」異動即時通一站式整合申請服務，可能要求申請人須持有不動產才能申請，建議申請前先向各縣市稅捐稽徵機關確認。

      👉各縣市稅籍異動即時通申請資訊請參考：
      基隆市 / 臺北市 / 新北市 / 桃園市 / 新竹縣 / 新竹市 / 苗栗縣 / 臺中市 / 彰化縣 / 南投縣 / 雲林縣 / 嘉義縣 / 嘉義市 / 臺南市 / 高雄市 / 屏東縣 / 花蓮縣 / 臺東縣 / 金門縣

       

      (二) 申請方式

       

      稅籍異動即時通提供2種申請方式：線上申請及臨櫃申請，以下整理各申請方式的申請資格、應備憑證/文件及相關說明，協助您快速了解，輕鬆挑選適合的申請方式 。

      申請方式 申請人 應備憑證/文件 說明
      線上申請 本人
      		可至地⽅稅網路申報作業系統，點選稅籍異動即時通透過應備憑證確認身分，線上填寫申請書後提交申請。
      👉使用手機線上申請，僅提供行動自然人憑證驗證方式。
      臨櫃申請 1. 本人
      2. 法定代理人
      		申請人攜帶應備文件，前往各縣市稅捐稽徵機關臨櫃辦理。

      ※ 注意事項：
      1. 申請後將寄發電子郵件驗證信，請於7日內完成驗證。如逾期未驗證或驗證失敗視同未完成申請，請重新申請並完成驗證。
      2. 申請完成後會寄送電⼦郵件申請完成通知信，確認收到後才算完成申請。
      3. 可一次申請全國縣市(連江縣除外)通知服務。

       

      (三) 常見問答

       

       
      💡申請稅籍異動即時通，要怎麼啟用LINE通知？

      要透過 LINE 收到稅籍異動即時通通知，需加入財政部地方稅LINE官方帳號為LINE好友，點擊會員中心申請「一般」會員並綁定「進階」會員，選擇訂閱想接收通知的縣市，完成驗證後，就能第一時間收到稅籍異動通知。

      💡申請稅籍異動即時通後，辦理什麼案件會收到通知？

      啟用稅籍異動即時通後，當名下不動產有土地增值稅或契稅申報行為時，系統會在案件「收件」時主動通知。

      💡稅籍異動即時通是否有適用期限，可以一次監控所有房產嗎？

      一經申請終身適用，若選擇開通全國通知服務，新購的土地或建物不用再次申請；若僅開通特定縣市通知服務，購買其他縣市的不動產則須重新申請。

      💡申請稅籍異動即時通可以通知我的家人嗎？

      稅籍異動即時通的通知對象僅限於申請人，因此申請時務必填寫本人的手機號碼及電子信箱。若希望增加接獲訊息機會，建議同時選擇多種通知管道，包含手機簡訊、電子郵件 及 LINE 推播通知（需綁定進階會員），以確保第一時間掌握異動訊息。

      💡換了手機號碼或電子郵件後，要修改稅籍異動即時通的通知方式嗎？

      若您的手機號碼或電子郵件有異動，請記得至稅籍異動即時通或稅捐稽徵機關臨櫃申請「資料變更」，才能正確收到通知。

      💡若申請人死亡，還會收到稅籍異動即時通的通知嗎？

      若申請人已死亡，稅捐稽徵機關將主動終止服務，不會再發送任何通知。

      二、地籍異動即時通

       

      地籍異動即時通是內政部地政司為保障不動產安全所推出的免費防詐通知服務，當名下土地或建物產生地籍異動（例如買賣、拍賣、贈與、配偶贈與、信託、抵押權設定、書狀補給、查封、假扣押、判決移轉、調解移轉、和解移轉、預告登記案件），於辦理情形為「收件」及「異動完成」時，地籍異動即時通將透過簡訊、電子郵件或兩者併行即時發送通知給已申請服務之登記名義人及指定一位受通知人，協助所有權人第一時間掌握資產狀況。

       

      (一) 申請資格

       

      地籍異動即時通主要是協助民眾防範不法詐騙案件，原則上應由土地或建物登記名義人本人申請。但考量實務上所有權人可能因年齡、身心狀況，或因工作、家庭因素不在國內而無法自行申請，可例外由他人代為申請。

      • 未成年人：由法定代理人（父母）申請，須提出法定代理證明文件。
      • 受監護宣告者：由法定代理人（監護人）申請，須提出法定代理證明文件。
      • 授權他人代辦者：由被授權人（代理人）申請，須提出經我國駐外館處或行政院設立（指定）之機構或委託之民間團體驗證、依法公證或認證的授權證明文件。

       

      (二) 申請方式

       

      地籍異動即時通有3種申請方式：線上申請、臨櫃申請及併土地登記案件申請，以下整理各申請方式的申請資格、應備憑證/文件及相關說明，協助您快速了解，輕鬆挑選適合的申請方式 。

      申請方式 申請人 應備憑證/文件 說明
      線上申請 土地或建物登記名義人
      		申請人可透過應備憑證登入數位櫃臺，點選地籍異動即時通的申請作業，線上填寫申請書向土地或建物所在地之登記機關提出申請。
      👉線上申請僅由不動產所在地的管轄機關受理，無法跨區辦理，但可一次產製不同轄區的申請書分別送出。
      臨櫃申請 1. 土地或建物登記名義人
      2. 法定代理人
      3. 被授權人
      		申請人可攜帶應備文件，前往任一地政事務所臨櫃辦理。
      👉如果申請的土地或建物分屬不同直轄市或縣（市）登記機關管轄，則需依照同一直轄市或縣（市）為單位，分別填寫申請書；可在不同縣市的地政事務所辦理申請，若標的屬於其他縣市，則會以內部轉送方式，轉由該縣市的登記機關處理。
      併土地登記案件申請 1. 土地或建物登記名義人（新取得）
      2. 法定代理人
      3. 被授權人
      		申請人在辦理土地登記時，可一併申請地籍異動即時通服務。
      👉申請書所蓋用印章或簽名，和申請土地登記之印章或簽名一致，就不用再核對申請人等身分。
      👉申請服務範圍限於該登記案件內土地及建物所載之登記機關轄區。
      👉如果申請的土地或建物分屬不同直轄市或縣（市）登記機關管轄，則需依照同一直轄市或縣（市）為單位，分別填寫申請書；若標的屬於其他縣市，則會以內部轉送方式，轉由該縣市的登記機關處理。

       

      (三) 常見問答

       

       
      💡申請地籍異動即時通後，辦理什麼案件會收到通知？

      當申辦以下登記案件時，地籍異動即時通系統會在「收件」以及「異動完成」時自動通知義務人或權利人（及指定受通知對象）！

      • 通知義務人的登記案件：買賣、拍賣、贈與、配偶贈與、信託、判決移轉、調解移轉、和解移轉、查封、假扣押、抵押權設定、預告登記。
      • 通知權利人的登記案件：書狀補給。
      💡申請地籍異動即時通後，會收到哪些通知？

      啟用地籍異動即時通服務後，系統會在兩個階段自動通知登記名義人：

      • 收件通知：當地政事務所收到有人申請變更不動產的登記（例如想要轉移所有權或設定抵押權）時，系統會立刻發送通知（手機簡訊或電子郵件）給所有權人，這時變更還沒完成，只是有人提出申請，讓所有權人提早發現異常並採取行動。
      • 異動完成通知：當地政事務所完成登記手續後，系統會再發送通知到手機簡訊或電子郵件，告知所有權人不動產的權利狀態已經正式異動。
      💡地籍異動即時通啟用後，可以一次監控所有房產嗎？

      申請時，申請人須填明名下房產所在縣市及行政區（或管轄登記機關）。同一縣市房產可向該縣市任一登記機關申辦，只需申請一次，即可一次監控名下在該縣市內全部房產。
      不過，若您於申辦後在未曾有房產之行政區首次取得新房產，仍需向登記機關申辦，才會對新取得房產啟動地籍異動即時通服務。

      💡申請地籍異動即時通可以通知我的家人嗎？

      為了增加接獲訊息機會，申請人可填寫最多2組（含指定通知對象，如家人、朋友）的手機號碼或電子郵件信箱。

      💡換了手機號碼或電子郵件後，要修改地籍異動即時通的通知方式嗎？

      若您換了手機號碼或電子郵件，地籍異動即時通的通知方式也要更新，請至數位櫃臺網路申請或至地政事務所臨櫃申請地籍異動即時通的「資料變更」，當處理完成後，才能正確收到通知。

      💡若申請人死亡，還會收到地籍異動即時通的通知嗎？

      若申請人已死亡，地政機關得主動終止服務，不再發送任何通知。

      站內相關服務連結：

      • 地籍異動即時通服務申請 土地或建物權利被申請買賣、拍賣、贈與、夫妻贈與、信託、抵押權設定、書狀補給、查封、假扣押、判決移轉、調解移轉、和解移轉案件，於辦理情形為「收件」及「異動完成」時，系統將透過簡訊或電子郵件主動發送通知給申請本項便民服務之所有權人。
      • 新北市-地籍異動即時通服務申請 不動產登記名義人本人或其法定代理人，可攜帶地籍異動即時通服務申請表及身分證明文件，前往任一地政事務所臨櫃辦理；或可由不動產登記名義人持自然人憑證或工商憑證線上申辦。
      • 臺中市-地籍異動即時通服務申請 不動產登記名義人本人或其法定代理人，攜帶申請書及身分證明文件，前往任一地政事務所臨櫃辦理。

       

       

      三、雙重保障：稅籍 + 地籍異動即時通

       

      以下透過案例，帶您快速了解稅籍異動即時通及地籍異動即時通在通知上的差異。建議兩項服務都要申請，才能為自身財產安全建立雙重防護，即時掌握不動產是否有異動，降低詐騙冒用或不當移轉的風險。

       

      案例 說明 稅籍
      異動即時通      		 地籍
      異動即時通
      不動產
      「買賣、贈與」      		 不動產「買賣、贈與」前，需先申報土地增值稅及契稅，此時稅籍異動即時通會發送通知。完成稅款繳納送地政機關辦理登記時，地籍異動即時通也會發送通知。
      👉土地增值稅是針對土地所有權人之土地，於移轉時因自然漲價所課徵的稅。
      👉因買賣、承典、交換、贈與、分割或因占有而依法取得所有權之不動產，需課徵契稅。      		 ✔️ ✔️
      不動產
      「抵押權設定、信託、預告登記、書狀補給、查封、假扣押」      		 不動產「抵押權設定、信託、預告登記、書狀補給」不須申報土地增值稅及契稅，稅籍異動即時通不會發送通知，但設定抵押權需要到地政機關辦理登記，因此地籍異動即時通會發送通知。另外地政機關受理法院囑託「查封、假扣押」時，地籍異動即時通也會發送通知。   ✔️
      未保存登記建物
      「買賣、贈與」      		 未保存登記建物因沒有建物所有權狀，「買賣、贈與」時不用向地政機關辦理登記，無法接到地籍異動即時通通知，但雙方當事人仍須向房屋所在地之稅捐稽徵機關申報契稅，因此稅籍異動即時通會發送通知。
      👉現行法令未強制規定建築完工後須向地政機關辦理建物所有權第一次登記（採任意登記制），若沒有辦理登記，就無法取得建物所有權狀，該建物即稱為未保存登記建物。      		 ✔️

      站內相關服務連結：
